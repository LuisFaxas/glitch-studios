**Step 5: If ANY overflow assertion fails, FIX THE PAGE.**

For each failing page:
1. Read the screenshot to identify the overflow source
2. Read the page component to find the offending element
3. Apply targeted CSS fix (typical fixes):
   - Container overflow: add `overflow-x-hidden` or `max-w-full`
   - Text overflow: add `truncate`, `break-words`, or `overflow-hidden`
   - Image overflow: add `max-w-full h-auto`
   - Table/grid overflow: add `min-w-0` on flex children, or `overflow-x-auto` wrapper
   - Absolute positioned elements: constrain with `max-w-screen` or `right-0`
4. Re-run ONLY the failing test to confirm the fix
