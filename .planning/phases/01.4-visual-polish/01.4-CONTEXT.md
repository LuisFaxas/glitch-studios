# Phase 1.4: Visual Polish & Sidebar Overhaul

## Goal

Fix every visual defect identified in the 2026-03-25 audit so the site looks intentional, not broken. The sidebar must feel like a living control panel, the home page must make a first impression, hover effects must be visible, and every page must follow the Cyberpunk Metro design language consistently.

**Audit reference:** .planning/VISUAL-AUDIT-2026-03-25.md
**Design spec:** .planning/DESIGN-LANGUAGE.md
**Current score:** 4/10
**Target score:** 8/10

## Depends On

Phase 1.3 (Supabase DB Driver Fix)

## Requirements Addressed

- INFR-03: Mobile-first responsive design across all pages with desktop optimization
- INFR-04: Cyberpunk/glitch visual identity — flat black & white, futuristic, sleek, dramatic typography
- INFR-05: Framer Motion page transitions and UI animations throughout

## Success Criteria (what must be TRUE)

1. Logo tile is visible (white-on-dark or inverted) with a noticeable warm glow that anchors the sidebar
2. Sidebar nav tiles use horizontal layout (icon left, label right) — not stacked vertically
3. Sidebar padding is tight (12px or less), all content (nav + widgets + sign-in) fits on a 900px viewport without scrolling OR a visible scroll hint exists
4. Sidebar has a collapsible icon-only compact mode (~64px) with toggle, or responsive breakpoints
5. Hovering any tile triggers a VISIBLE glitch distortion on the tile content — not just an invisible overlay
6. Home page hero is visible (GlitchLogo text + CTA buttons readable against placeholder) and empty DB sections either show fallback content or are hidden
7. /beats shows a "Coming Soon" placeholder page (not 404); About tile renamed to "Artists" or routes correctly
8. Mobile nav overlay renders visible content (nav tiles, widgets, logo)
9. Footer restyled to monochrome hex palette, sharp corners, mono font headings — matches Cyberpunk Metro
10. Contact form inputs are flat (0px radius, #333 border, white focus border), submit button is inverted tile (white bg, black text)
11. Services mobile accordion items wrapped in tile-styled containers
12. Blog cards handle missing images gracefully (pattern/gradient placeholder or text-only layout)

## Scope Boundaries

**In scope:**
- All issues from VISUAL-AUDIT-2026-03-25.md (C1-C9, M1-M6, P1-P6, N1-N2, N4)
