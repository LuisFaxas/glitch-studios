    2. Rewrite `src/lib/db.ts` to use the Neon HTTP adapter. Replace the entire file with:
       ```typescript
       import { neon } from "@neondatabase/serverless"

       import * as schema from "@/db/schema"

       const sql = neon(databaseUrl)
       export const db = drizzle(sql, { schema })
       ```

       Key differences from old code:
       - `neon()` creates an HTTP-only client (no persistent TCP connection)
       - `drizzle-orm/neon-http` adapter instead of `drizzle-orm/postgres-js`
