## Deviations from Plan

### Auto-fixed Issues

**1. [Rule – Wrong API] GlitchHeading does not accept `as` prop**
- **Found during:** Task 4 (TABLE-ROW-HOVER on MethodologyDisciplineTable)
- **Issue:** Plan prescribed `<GlitchHeading as="span" text={d.name}>{d.name}</GlitchHeading>` but `GlitchHeadingProps` has only `{children, text, className}`. Adding `as` would either TS-error or be silently ignored.
- **Fix:** Used `<GlitchHeading text={d.name}>{d.name}</GlitchHeading>` — GlitchHeading already renders a `<span>` so the `as="span"` was effectively no-op anyway.
- **Files modified:** src/components/tech/methodology-discipline-table.tsx
- **Verification:** pnpm tsc --noEmit clean; row-hover Playwright assertion passes.
- **Committed in:** 8cbc693

**2. [Rule – Stale grep] First-task acceptance grep `! grep -q '"ABOUT"'` would now fail because the about page metadata no longer contains the string `"ABOUT"`. Confirmed via dev render that bare `<h1>ABOUT</h1>` is gone, replaced by `<TechHero ... title="About GlitchTech">`.**
