**Framer Motion Entrance Animation:**
```typescript
initial={{ scale: 0.8, opacity: 0 }}
animate={{ scale: 1, opacity: 1 }}
transition={{
  duration: 0.6,
  ease: [0.215, 0.61, 0.355, 1], // cubic-bezier matching UI-SPEC
}}
```
Duration: 600ms, easing: cubic-bezier(0.215, 0.61, 0.355, 1) per UI-SPEC.

**Post-Entrance Glitch Trigger:**
- On animation complete (600ms), immediately trigger a single glitch-hover animation (200ms)
- Use Framer Motion's onAnimationComplete callback
- Apply a CSS class that plays the glitch-hover keyframe once
- Effect: Logo "powers on" with scale+opacity fade, then glitches to "lock in"

**Styling:**
- Background: url(/Untitled-2.png) center / cover
- Glow: box-shadow 0 0 15px rgba(255,250,240,0.15)
- Border: 1px solid #222 (default state)
- Padding: 16px
- Border-radius: 0px (sharp)
- Hover state: border #444 + glitch-hover animation (same as any Tile)
- Background color: #111 (behind image)

**Tile States:**
- Default: No hover effect while animating in
- Hover: glitch-hover animation + border brightens to #444
- Active: N/A (logo is not a nav target, never "active")

**Implementation:**
Use motion.div from "motion/react" for entrance animation. Include an onAnimationComplete handler to trigger post-entrance glitch. Apply box-shadow glow and background image via style or className.

**Accessibility:**

- aria-label="Glitch Studios"
- Not keyboard-focusable (decorative)
- Not a navigation target
