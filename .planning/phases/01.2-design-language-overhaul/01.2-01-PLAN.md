```css
  20%  { clip-path: inset(20% 0 60% 0); transform: translate(-2px, 0); }

  100% { clip-path: inset(0 0 0 0); transform: translate(0); }
}
```
Duration: 200ms, timing: steps(1), iteration: 1, trigger: hover.

**Add @keyframes scan-line:**
```css
@keyframes scan-line {
  0%   { top: -1px; opacity: 0; }
  10%  { opacity: 1; }
  90%  { opacity: 1; }
  100% { top: 100%; opacity: 0; }
}
```
Duration: 400ms. Renders as 1px white line (rgba(255,255,255,0.1)) sweeping top-to-bottom via ::after pseudo-element.

**Add Tailwind animation utilities:**
In @layer utilities, add:
```css
@layer utilities {
  .animate-glitch-hover {
    animation: glitch-hover 200ms steps(1);
  }
  .animate-scan-line {
    animation: scan-line 400ms ease-out forwards;
  }
}
```

**Add reduced-motion support:**
Wrap glitch animations in @supports and @media prefers-reduced-motion:
```css

    /* glitch animations here */
  }
  @media (prefers-reduced-motion: reduce) {
    .animate-glitch-hover {
      animation: none;
      opacity: 1;
    }
  }
}
```

  <action>
Create `src/components/tiles/tile.tsx` as a reusable foundational tile component.

**TypeScript Interface:**
```typescript
export type TileSize = "small" | "medium" | "wide" | "large"

export interface TileProps {
  size?: TileSize
  label?: string
  sublabel?: string
  icon?: React.ReactNode
  isActive?: boolean
  href?: string
  onClick?: () => void
  children?: React.ReactNode
  className?: string
}
```

**Size Grid Mappings (CSS Grid):**
- `small`: col-span-1 row-span-1 aspect-square (1x1 equal width/height)
- `medium`: col-span-2 row-span-1 (2x1 primary nav size)
- `wide`: col-span-2 row-span-1 (2x1 full sidebar width, same span as medium but used for Contact, widgets)
- `large`: col-span-2 row-span-2 (2x2 logo and featured content)

**Default State Styling:**
- Background: #111111 (gray-950 or custom bg-gray-950)
- Border: 1px solid #222222 (gray-800)
- Text: #f5f5f0 (off-white, use text-neutral-100 or custom)
- Padding: 16px (p-4)
- Border-radius: 0px (sharp corners)
- Transition: transition-colors duration-200

**Active State (isActive={true}):**
- Background: #f5f5f0 (inverted white)
- Border: 1px solid #f5f5f0
- Text: #000000 (black)
- Glow: box-shadow 0 0 20px rgba(255,255,255,0.08)
- NO hover animation (locked in)
- aria-current="page"

**Hover State (when not active):**
- Background: #1a1a1a (subtle lift)
- Border: 1px solid #444444 (brightened)
- Apply: animate-glitch-hover (triggers 200ms glitch animation from globals.css)
- Include ::after pseudo-element for scan-line animation
- Cursor: pointer

**Icon Positioning:**
- Lucide icon, 20px size
- Positioned top-left within the tile
- For large tiles: icon centered top, label below with breathing room
- Use flex layout: flex flex-col items-start justify-start gap-2

**Label Typography:**
- Font: JetBrains Mono (if available), fallback mono stack
- Size: 18px for nav tiles
- Weight: 700 (bold)
- Case: uppercase
- Letter-spacing: 0.05em

**Sublabel (optional):**
- Font: Inter, 13px, 400 weight
- Color: #888888 (gray-500 or --color-text-muted)
- Appears below label when provided

**Accessibility Requirements:**
- All tiles must be keyboard-focusable with visible focus indicator
- Focus outline: 1px #f5f5f0, 2px offset
- Semantics: Render as <Link> if href provided, otherwise <button> or <div>
- ARIA: aria-current="page" on active tile
- Tab order: natural DOM order
- Mobile touch targets: 48px minimum (achieved via grid sizing)

**Scan-line Overlay (::after pseudo-element):**
- Only visible on hover (when not active)
- 1px height, 100% width
- Opacity: 0.1 (rgba(255,255,255,0.1))
- Animation: animate-scan-line (defined in Task 1)
- Positioned: absolute, top to bottom sweep

**Component Exports:**
```typescript
export { Tile, type TileProps, type TileSize }
```

**Implementation Notes:**
- Use clsx for conditional class composition
- Do NOT use any color accents
- Do NOT add rounded corners beyond 0px
- Children prop allows widget-specific layouts (used by widget components)
- className prop allows one-off overrides if needed

**Example Usage (for Plan 02 reference):**
```typescript
<Tile
  size="medium"
  label="Services"
  icon={<Wrench size={20} />}
  href="/services"
  isActive={pathname === "/services"}
/>
```
