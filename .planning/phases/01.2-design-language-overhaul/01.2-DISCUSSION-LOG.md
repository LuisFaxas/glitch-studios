## Logo Tile Treatment

| Option | Description | Selected |
|--------|-------------|----------|
| PNG image only | Static PNG in tile, no animation | |
| CSS animation on load, then PNG | Dramatic CSS entrance, swap to PNG | |
| PNG with glitch hover | Static PNG, glitch on hover only | |
| Keep CSS text component | Ignore PNG, keep CSS animation | |
| **PNG with warm glow + glitch hover + dramatic entrance** | **User's custom choice** | ✓ |

**User's choice:** PNG with incandescent warm glow, Framer Motion dramatic entrance animation, and glitch hover. Custom combination of options.
**Follow-up:** Glow should be warm white (monochrome-safe) — `rgba(255,250,240,0.15)`. NOT true amber/gold. Stays within monochrome palette.
