**Props:** `{ beat: BeatSummary }`

**Structure (per UI-SPEC card anatomy):**
1. Outer div: `className="group border border-[#222] bg-[#111] hover:border-[#444] hover:bg-[#1a1a1a] transition-colors"`
2. Cover art section: `div.relative.aspect-square` containing:
   - If `beat.coverArtUrl`: Next.js `Image` with `fill`, `object-cover`, `sizes="(min-width:1024px)33vw,(min-width:768px)50vw,100vw"`
   - Fallback (no art): `div` with `bg-[#222] flex h-full w-full items-center justify-center` centered `Music` icon (24px, `text-[#555]`)
   - Play overlay button: `absolute inset-0 flex items-center justify-center bg-black/60 opacity-0 group-hover:opacity-100 transition-opacity duration-150`. Icon is `Play` or `Pause` from lucide-react at `h-12 w-12 text-white`. `aria-label` toggles between "Play {title}" and "Pause {title}".
   - **CRITICAL: Play overlay must be disabled when no preview audio.** If `!beat.previewAudioUrl`, do NOT render the play overlay button at all. This prevents confusion on beats without audio.
   - Play overlay uses `e.stopPropagation()` to prevent license modal from opening.
3. Info section: `div.p-4.cursor-pointer` with `onClick={() => setLicenseModalOpen(true)}`
   - Row 1: flex between title (`truncate font-mono text-[15px] font-bold text-[#f5f5f0] group-hover:glitch-hover`) and price (`shrink-0 font-mono text-[15px] font-bold text-[#f5f5f0]` prefixed with "FROM $")
   - Row 2: producer name (`block truncate font-sans text-[11px] text-[#888]`) -- use `beat.producers[0]?.name ?? "Glitch Studios"`. If multiple producers, still show only the first one (card space is limited).
   - Row 3: metadata badges (genre, BPM, key) -- use `Badge` component with `className="rounded-none border-[#333] bg-[#222] font-mono text-[11px] uppercase text-[#f5f5f0]"`, wrapped in `div.mt-2.flex.flex-wrap.gap-1`
   - Row 4: mood tags (per BEATS-03) -- if `beat.moods && beat.moods.length > 0`, render `span.mt-1.block.font-sans.text-[11px].text-[#888]` with `beat.moods.slice(0, 2).join(", ")`. If moods is null or empty array, render nothing.
   - **Price edge case:** If no active pricing tiers exist (`lowestPrice` is undefined), do NOT render the price span. Title takes full width.
   - **Sold-exclusive status:** If `beat.status === "sold_exclusive"`, render the card at 40% opacity: add `opacity-40` to the outer div and disable the info section onClick (no license modal).
4. LicenseModal: `<LicenseModal beat={beat} isOpen={licenseModalOpen} onClose={() => setLicenseModalOpen(false)} />`

**State:**
- `const [licenseModalOpen, setLicenseModalOpen] = useState(false)`
- `const { currentBeat, isPlaying, play, pause } = useAudioPlayer()`
- `const isCurrentBeat = currentBeat?.id === beat.id`
- `const isActivePlaying = isCurrentBeat && isPlaying`
- Price: `const lowestPrice = beat.pricing.filter(p => p.isActive).sort((a, b) => Number(a.price) - Number(b.price))[0]`

**handlePlay function** (copy pattern from beat-row.tsx, including disabled check):
```typescript
function handlePlay(e: React.MouseEvent) {
  e.stopPropagation()
  if (!beat.previewAudioUrl) return
  if (isActivePlaying) { pause() }
  else { play({ id: beat.id, title: beat.title, artist: beat.producers[0]?.name ?? "Glitch Studios", previewAudioUrl: beat.previewAudioUrl, coverArtUrl: beat.coverArtUrl }) }
}
```
