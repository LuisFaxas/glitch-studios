**2. Create src/actions/admin-newsletter.ts** ("use server"):

- `getSubscribers(filters?: { search?: string, page?: number })`: requirePermission("send_newsletters")
  - Query newsletterSubscribers with optional email ILIKE search
  - Paginate: 50 per page
  - Return { subscribers, totalPages }

- `getSubscribersBySegment(segment: "all" | "beat_buyers" | "studio_clients")`: requirePermission("send_newsletters")
  - "all": all active subscribers
  - "beat_buyers": subscribers where tags array contains 'beat_buyer'
  - "studio_clients": subscribers where tags array contains 'studio_client'
  - Return { subscribers, count }

- `getSegmentCounts()`: requirePermission("send_newsletters")
  - Return { all: number, beat_buyers: number, studio_clients: number }

- `sendNewsletter(data: { subject: string, body: string, segment: "all" | "beat_buyers" | "studio_clients" })`: requirePermission("send_newsletters")
  - Get subscribers by segment
  - Render NewsletterBroadcastEmail to HTML using render() from @react-email/components
  - Chunk subscribers into batches of 100
  - For each batch: call `resend.batch.send()` with array of email objects
  - After all batches sent: insert into newsletterBroadcasts table { subject, body, segment, recipientCount, sentAt: now, sentBy: session.user.name }
  - Return { sent: recipientCount }
  - On error: throw with message "Newsletter could not be sent. Check your Resend configuration and try again."

- `getBroadcasts(page?: number)`: requirePermission("send_newsletters")
  - Query newsletterBroadcasts ordered by createdAt desc
  - Paginate: 20 per page
  - Return { broadcasts, totalPages }

- `removeSubscriber(subscriberId: string)`: requirePermission("send_newsletters")
  - Delete from newsletterSubscribers

- `autoTagSubscribers()`: utility function (called after purchases/bookings)
  - Query users with orders -> tag their newsletter subscription as 'beat_buyer'
  - Query users with bookings -> tag their newsletter subscription as 'studio_client'
  - UPDATE newsletterSubscribers SET tags = computed_tags WHERE email = user.email
