**1. Create src/actions/admin-homepage.ts** ("use server"):

- `getHomepageSections()`: requirePermission("manage_settings")
  - Query homepageSections ordered by sortOrder
  - If empty (first load), seed default sections:
    - { sectionType: "hero", sortOrder: 0, isVisible: true, config: '{"title":"GLITCH STUDIOS","subtitle":"Music & Video Production","ctaText":"Book Now","ctaLink":"/booking"}' }
    - { sectionType: "featured_beats", sortOrder: 1, isVisible: true, config: '{"beatIds":[]}' }
    - { sectionType: "services", sortOrder: 2, isVisible: true, config: '{}' }
    - { sectionType: "testimonials", sortOrder: 3, isVisible: true, config: '{}' }
    - { sectionType: "blog", sortOrder: 4, isVisible: true, config: '{}' }
    - { sectionType: "portfolio", sortOrder: 5, isVisible: true, config: '{}' }
  - Return sections array

- `updateSectionOrder(orderedIds: string[])`: requirePermission("manage_settings")
  - For each id at index i: UPDATE homepageSections SET sortOrder=i WHERE id=orderedIds[i]
  - Toast: "Changes saved."

- `updateSectionVisibility(sectionId: string, isVisible: boolean)`: requirePermission("manage_settings")
  - UPDATE homepageSections SET isVisible WHERE id=sectionId

- `updateSectionConfig(sectionId: string, config: string)`: requirePermission("manage_settings")
  - UPDATE homepageSections SET config, updatedAt WHERE id=sectionId
  - Config is JSON string with section-specific data:
    - hero: { title, subtitle, ctaText, ctaLink, backgroundMediaUrl }
    - featured_beats: { beatIds: string[] }
    - featured_videos: { portfolioItemIds: string[] }
    - services, testimonials, blog, portfolio: {} (auto-rendered, no config needed)

- `getAvailableBeats()`: query beats WHERE status='published' (for featured beats picker)
- `getAvailablePortfolioItems()`: query portfolioItems (for featured videos picker)
