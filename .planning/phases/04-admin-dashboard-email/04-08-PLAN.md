**1. Create src/actions/admin-homepage.ts** ("use server"):

- `getHomepageSections()`: requirePermission("manage_settings")
  - Query homepageSections ordered by sortOrder
  - If empty (first load), seed default sections:
    ```typescript
    const defaults = [
      { sectionType: "hero", sortOrder: 0, isVisible: true, config: JSON.stringify({ title: "GLITCH STUDIOS", subtitle: "Music & Video Production", ctaText: "Book Now", ctaLink: "/booking" }) },
      { sectionType: "featured_beats", sortOrder: 1, isVisible: true, config: JSON.stringify({ beatIds: [] }) },
      { sectionType: "services", sortOrder: 2, isVisible: true, config: "{}" },
      { sectionType: "portfolio", sortOrder: 3, isVisible: true, config: "{}" },
      { sectionType: "testimonials", sortOrder: 4, isVisible: true, config: "{}" },
    ]
    await db.insert(homepageSections).values(defaults)
    ```
  - Return sections array

- `getPublicHomepageSections()`: NO permission check (public route calls this)
  - Query homepageSections WHERE isVisible=true ordered by sortOrder
  - Return sections array (or empty array if none seeded yet)

- `updateSectionOrder(orderedIds: string[])`: requirePermission("manage_settings")
  - For each id at index i: UPDATE homepageSections SET sortOrder=i WHERE id=orderedIds[i]

- `updateSectionVisibility(sectionId: string, isVisible: boolean)`: requirePermission("manage_settings")
  - UPDATE homepageSections SET isVisible WHERE id=sectionId

- `updateSectionConfig(sectionId: string, config: string)`: requirePermission("manage_settings")
  - UPDATE homepageSections SET config, updatedAt WHERE id=sectionId
  - Config JSON structures by section type:
    - hero: `{ title: string, subtitle: string, ctaText: string, ctaLink: string, backgroundMediaUrl?: string }`
    - featured_beats: `{ beatIds: string[] }`
    - portfolio: `{ portfolioItemIds?: string[] }` (optional -- if empty, shows all)
    - services, testimonials, blog: `{}` (auto-rendered)

- `getAvailableBeats()`: query beats WHERE status='published', return id + title (for featured beat picker)
- `getAvailablePortfolioItems()`: query portfolioItems WHERE isActive=true, return id + title
