**1. Create src/actions/admin-media.ts** ("use server"):

- `uploadMedia(formData: FormData)`: requirePermission("manage_media")
  - Extract file metadata (filename, mimeType, size)
  - Generate key: `media/${Date.now()}-${slugifiedFilename}`
  - Call getUploadUrl from r2.ts for presigned URL
  - Return { uploadUrl, key, mediaId } -- client uploads directly to R2
  - After client confirms upload, call `confirmMediaUpload(key, url, metadata)` to insert into mediaAssets table

- `confirmMediaUpload(data: { key, filename, mimeType, size, width?, height?, duration?, alt? })`: requirePermission("manage_media")
  - Construct URL: `${process.env.R2_PUBLIC_URL}/${data.key}`
  - Insert into mediaAssets table
  - Return the created media asset

- `deleteMedia(mediaId: string)`: requirePermission("manage_media")
  - Delete from mediaAssets table
  - Optionally delete from R2 (use DeleteObjectCommand)

- `getMediaAssets(filters?: { type?: string, search?: string, page?: number })`:
  - Query mediaAssets with optional mimeType LIKE filter and filename ILIKE search
  - Paginate: 24 items per page
  - Return { items, totalPages, currentPage }
