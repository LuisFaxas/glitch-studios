**1. Create src/actions/admin-inbox.ts** ("use server"):

- `getMessages(filters?: { page?: number })`: requirePermission("reply_messages")
  - Query contactSubmissions ordered by createdAt desc
  - Include reply count per submission (LEFT JOIN contactReplies)
  - Paginate: 20 per page
  - Return { messages, totalPages, unreadCount }

- `getMessage(submissionId: string)`: requirePermission("reply_messages")
  - Fetch contactSubmission by ID
  - Fetch all contactReplies for this submission ordered by sentAt asc
  - Mark as read: UPDATE contactSubmissions SET isRead=true WHERE id=submissionId
  - Return { submission, replies }

- `replyToMessage(submissionId: string, body: string)`: requirePermission("reply_messages")
  - Get the contact submission for sender email
  - Insert into contactReplies table: { submissionId, body, sentBy: session.user.name, sentAt: now }
  - Send reply email via Resend:
    ```typescript
    await resend.emails.send({
      from: "Glitch Studios <noreply@glitchstudios.com>",
      to: submission.email,
      subject: `Re: ${submission.service || "Your message"} - Glitch Studios`,
      text: body
    })
    ```
  - Toast: "Reply sent to {email}."

- `getUnreadCount()`: COUNT contactSubmissions WHERE isRead=false (used by AdminShell for sidebar badge)
