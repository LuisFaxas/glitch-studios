1. Create `src/components/cart/cart-provider.tsx` — "use client" React context:

Per Research Pattern 3 and D-14:

```typescript
"use client"
import { createContext, useContext, useState, useEffect, useCallback, useMemo } from "react"
import type { CartItem } from "@/types/beats"

interface CartState {
  items: CartItem[]
  isOpen: boolean
  isMounted: boolean
  addItem: (item: CartItem) => void
  removeItem: (beatId: string, licenseTier: string) => void
  clearCart: () => void
  openCart: () => void
  closeCart: () => void
  toggleCart: () => void
  total: number
  itemCount: number
}
```

Key implementation details:
- `isMounted` flag: Start as `false`, set to `true` in useEffect. Prevents hydration mismatch on cart badge (per Research Pitfall 4).
- `isOpen` state: Controls the cart drawer visibility.
- localStorage key: `"glitch-cart"`
- Hydrate from localStorage in useEffect (client-only), NOT in useState initializer.
- Persist to localStorage in a separate useEffect watching `items`.
- `addItem`: Check if same beatId + licenseTier already exists — if so, don't add duplicate.
- `removeItem`: Filter by both beatId AND licenseTier (user might have same beat with different tiers).
- `total`: Computed via `useMemo` summing all item prices.
- `itemCount`: `items.length`
