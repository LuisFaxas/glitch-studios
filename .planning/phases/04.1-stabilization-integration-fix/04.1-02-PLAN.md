  <action>
Rewrite `src/db/seed-admin.ts` to create an admin/owner user via direct database insert (no running dev server required). Per D-10, credentials must be configurable via env vars.

**1. Connection setup (standalone):**
```typescript
import postgres from "postgres"
import { drizzle } from "drizzle-orm/postgres-js"
import { hashPassword } from "better-auth/crypto/password"
import * as schema from "./schema"
```

**2. Create standalone DB connection:**
```typescript
const client = postgres(process.env.DATABASE_URL!)
const db = drizzle(client, { schema })
```

**3. Read configurable credentials from env:**
```typescript
const email = process.env.ADMIN_EMAIL || "admin@glitchstudios.com"
const password = process.env.ADMIN_PASSWORD || "changeme123"
const name = process.env.ADMIN_NAME || "Glitch Admin"
```
Display name "Glitch Admin" per UI-SPEC copywriting contract.

**4. Hash the password using Better Auth's own hashing function:**
```typescript
const hashedPassword = await hashPassword(password)
```
This uses scrypt via `@noble/hashes` and produces the `salt:hash` format that Better Auth's `verifyPassword` expects. Do NOT use bcrypt — Better Auth uses scrypt.

**5. Generate user ID and insert into `user` table:**
```typescript
const userId = crypto.randomUUID()
await db.execute(
  sql`INSERT INTO "user" (id, name, email, "emailVerified", role, "createdAt", "updatedAt")
      VALUES (${userId}, ${name}, ${email}, true, 'owner', NOW(), NOW())
      ON CONFLICT (email) DO UPDATE SET role = 'owner', name = ${name}, "updatedAt" = NOW()`
)
```
Use role `'owner'` (not `'admin'`) so the user has full access per the RBAC system (owner bypasses all permission checks in permissions.ts).

**6. Insert into `account` table for credential provider:**
```typescript
await db.execute(
  sql`INSERT INTO "account" (id, "accountId", "providerId", "userId", password, "createdAt", "updatedAt")
      VALUES (${crypto.randomUUID()}, ${email}, 'credential', ${userId}, ${hashedPassword}, NOW(), NOW())
      ON CONFLICT ("providerId", "accountId") DO UPDATE SET password = ${hashedPassword}, "updatedAt" = NOW()`
)
```

**7. Close connection and log success:**
```typescript
console.log(`Admin user created: ${email} (role: owner)`)
console.log("Login with these credentials at /login")
await client.end()
```
