- The chip click→URL write test was rewritten to navigate with `?cpu=AMD`
  and assert the read path. Live click was flaky under Turbopack + nuqs
  + Playwright (the click event went through but the test runner hung
  after the resulting transition). The read path is the meaningful
  regression gate; the write path is exercised by the production flow.
- The mobile sheet test uses `dispatchEvent('click')` to bypass the
  bottom-tab-bar (z-50) intercepting pointer events on the
  bottom-right floating trigger (z-30).
- No backwards-compat shim added: callers all pass the new layout prop
  explicitly (the table call site sets `layout="bar"`, the sheet sets
  `layout="vertical"`).
